# 🌦 Weather App

A responsive and modern Weather Application built with React.js that shows real-time weather data using the OpenWeather API.

🚀 Features
- Search weather by city name
- Real-time API data fetching
- Displays temperature in Celsius
- Shows weather conditions (Clear, Clouds, Rain, etc.)
- Responsive design (Mobile + Desktop)
- Simple and clean UI

🛠 Tech Stack
- React.js
- JavaScript (ES6+)
- CSS3
- Vite
- OpenWeather API
